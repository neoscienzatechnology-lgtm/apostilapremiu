## 🛠️ Para Desenvolvedores

### Instalação Local

```bash
# Clone
git clone https://github.com/SEU_USUARIO/apostila-javascript.git
cd apostila-javascript

# Instale dependências
npm install

# Build
npm run build

# Abra no navegador
start index.html  # Windows
open index.html   # Mac
xdg-open index.html  # Linux
```

### Desenvolvimento

```bash
# Modo watch (CSS)
npm run dev

# Servidor local
npm run serve

# Acesse: http://localhost:8000
```

### Deploy

```bash
# Automático
npm run deploy:manual

# Ou use o script
.\deploy.ps1  # Windows
bash deploy.sh  # Linux/Mac
```

## 📁 Estrutura

```
apostila-javascript/
├── index.html                    # Página inicial
├── apostila-javascript-premium.html  # Apostila completa
├── css/
│   └── output.css               # CSS compilado
├── js/
│   ├── app.js                   # App principal
│   ├── sanitize.js              # Segurança XSS
│   ├── playground.js            # Playground seguro
│   ├── progress-manager.js      # Backup/restore
│   ├── notes-system.js          # Notas pessoais
│   ├── analytics.js             # Analytics local
│   └── accessibility.js         # Acessibilidade
├── manifest.json                # PWA config
└── sw.js                        # Service Worker
```
