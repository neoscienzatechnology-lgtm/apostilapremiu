## 🆘 Problemas?

### Site mostra 404
- Aguarde mais 2 minutos
- Limpe cache: Ctrl+Shift+R
- Verifique se GitHub Pages está ativado

### CSS não carrega
- Verifique se `css/output.css` existe no repositório
- Execute `npm run build` e faça novo push

### Mudanças não aparecem
- Limpe cache do navegador
- Abra em modo anônimo (Ctrl+Shift+N)
- Aguarde 1-2 minutos
